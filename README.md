# Gototrip – Landing (Next.js + Tailwind)

## Démarrer
```bash
npm install
npm run dev
```

Puis ouvre http://localhost:3000

## Stack
- Next.js 14 (app router)
- Tailwind CSS
- framer-motion + lucide-react (icônes)

## Personnalisation
- Remplace `public/logo.png` par ton logo final.
- Le style est dans `app/styles/globals.css`.
- La page d’accueil est `app/page.tsx`.
